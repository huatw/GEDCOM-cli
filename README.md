# GEDCOM-cli

## Usage

```bash
# clone project
git clone <..> 

# download dependencies
yarn install # or npm install

# parse specific file
yarn start parse -f gedcom/test.ged
# parse all files under specific directory
yarn start parse -d gedcom

# see helper for usage
yarn start -h 

# run unit test
yarn run test
```
